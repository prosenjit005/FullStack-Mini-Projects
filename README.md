# FullStack-Mini-Projects




## 1. P1-SpringBoot-MySQL
SpringBoot Project with local MySQL integration.  
  
Initial Dependencies added:  
    1. Lombok  
    2. Spring Data JPA  
    3. MySQL Driver  
    4. Spring Web  

CRUD Operations Done here:  
    1. addCompany  
    2. addCompanies  
    3. findAllCompanies  
    4. findCompanyById  
    5. findCompanyByCode  
    6. updateCompany  
    7. deleteCompany
